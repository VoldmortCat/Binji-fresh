# 斌记鲜购

斌记鲜购是一个线上生鲜百货商城，主营瓜果蔬菜、饮料零食、日用食材等，用户通过微信小程序即可在线选购下单，配送上门。同时为商家提供 Web 端管理后台，方便运营商品、处理订单和查看数据。

## 项目组成

| 模块 | 目录 | 技术栈 | 说明 |
|------|------|--------|------|
| C 端小程序 | `mp-weixin/` | uni-app + 微信小程序原生 | 消费者选购前端，覆盖商品浏览、加购下单、在线支付、订单追踪全流程 |
| 管理后台 | `binji-admin/` | Vue 2 + TypeScript + Element UI | 商家运营后台，支持商品/分类/套餐管理、订单处理、员工管理、数据报表 |
| 后端服务 | `binji-fresh/` | Spring Boot 3 + MyBatis + Redis + MySQL | 提供全部 API 接口，包含 admin 端和 user 端两套鉴权体系 |

## 主要功能

### C 端（微信小程序）

- **商品浏览** — 分类导航、热销推荐，支持规格选择和详情查看
- **购物车** — 多商品加购，数量增减，一键清空
- **下单支付** — 地址管理、订单备注、微信支付（含 15 分钟超时自动取消）
- **订单中心** — 历史订单、状态追踪、再来一单、催单
- **AI 智能导购助手** — 基于 DeepSeek 大模型的对话式商品推荐，可直接从推荐卡片加入购物车
- **地址管理** — 支持多标签（公司/家/学校/其他），省市县联动选择

### 管理后台（Web）

- **商品管理** — 商品的新增、编辑、下架、规格配置
- **套餐管理** — 组合套餐的新增、编辑、删除、启停、套餐商品关联
- **分类管理** — 商品/套餐分类的新增、编辑、删除、启用禁用
- **订单管理** — 订单查询、接单、拒单、配送、完成，支持条件搜索
- **员工管理** — 员工账号的新增、编辑、启用禁用、密码重置
- **数据报表** — 营业额统计、用户统计、订单统计、销量 Top10、Excel 导出
- **工作台** — 今日运营数据概览（营收、订单、商品/套餐数量）

### 后端特性

- JWT 双端鉴权（admin / user）
- Redis 缓存商品列表、营业状态、热销商品
- WebSocket 来单实时提醒
- Spring AI 集成 DeepSeek 大模型 + DashScope 文本嵌入
- 定时任务：订单超时处理、热销商品刷新
- Swagger/Knife4j API 文档
