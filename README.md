# 斌记鲜购

斌记鲜购是一个集线上点餐、外卖配送、智能推荐于一体的生鲜外卖平台，面向 C 端消费者提供微信小程序点餐服务，同时为商家提供 Web 端管理后台。

## 项目组成

| 模块 | 目录 | 技术栈 | 说明 |
|------|------|--------|------|
| C 端小程序 | `mp-weixin/` | uni-app + 微信小程序原生 | 消费者点餐前端，覆盖浏览菜品、下单、支付、订单追踪全流程 |
| 管理后台 | `project-sky-admin-vue-ts/` | Vue 2 + TypeScript + Element UI | 商家运营管理后台，支持菜品/套餐/分类管理、订单处理、员工管理、数据报表 |
| 后端服务 | `binji-fresh/` | Spring Boot 3 + MyBatis + Redis + MySQL | 提供全部 API 接口，包含 admin 端和 user 端两套鉴权体系 |

## 主要功能

### C 端（微信小程序）

- **菜品浏览** — 分类导航、热销推荐，支持规格选择和详情查看
- **购物车** — 多菜品加购，数量增减，一键清空
- **下单支付** — 地址管理、订单备注、微信支付（含 15 分钟超时自动取消）
- **订单中心** — 历史订单、状态追踪、再来一单、催单
- **AI 智能点餐助手** — 基于 DeepSeek 大模型的对话式菜品推荐，可直接从推荐卡片加入购物车
- **地址管理** — 最多 4 个标签（公司/家/学校/其他），支持省市县联动选择

### 管理后台（Web）

- **菜品管理** — 菜品的新增、编辑、删除、启停、口味配置
- **套餐管理** — 套餐的新增、编辑、删除、启停、套餐菜品关联
- **分类管理** — 菜品/套餐分类的新增、编辑、删除、启用禁用
- **订单管理** — 订单查询、接单、拒单、派送、完成，支持条件搜索
- **员工管理** — 员工账号的新增、编辑、启用禁用、密码重置
- **数据报表** — 营业额统计、用户统计、订单统计、销量 Top10、Excel 导出
- **工作台** — 今日运营数据概览（营收、订单、菜品/套餐数量）

### 后端特性

- JWT 双端鉴权（admin / user）
- Redis 缓存菜品列表、营业状态、热销菜品
- WebSocket 来单实时提醒
- 阿里云 OSS 文件上传
- 微信支付 V3 集成
- Spring AI 集成 DeepSeek 大模型 + DashScope 文本嵌入
- 定时任务：订单超时处理、热销菜品刷新
- Swagger/Knife4j API 文档
